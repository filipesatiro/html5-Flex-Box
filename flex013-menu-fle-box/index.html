Teste de Flexbox

Flexbox é um modelo de layout do CSS criado para facilitar a organização de elementos dentro de um container, principalmente quando se trata de alinhamento e distribuição de espaço. Ele permite que os itens se ajustem automaticamente ao tamanho da tela, tornando o design mais responsivo e adaptável.

Com o Flexbox, é possível controlar a direção dos elementos, alinhar no centro, nas extremidades ou distribuir igualmente com poucas linhas de código. Isso torna o desenvolvimento mais rápido e eficiente, especialmente para criar interfaces modernas e organizadas.
